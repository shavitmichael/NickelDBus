Detailed Description

The NDB class registers a service on d-bus of Kobo e-readers.

NDB provides a bridge between Kobo's proprietary software, libnickel, and other software running on Kobo e-readers. It registers itself as a service on d-bus, and provides methods and signals to monitor and interact with nickel.
